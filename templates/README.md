# Word Templates

Place your `.docx` templates in this folder.

Supported filenames matches the dropdown options:
- `UIIC.docx`
- `NIA.docx`
- `OIC.docx`
- ...

Fallback: `template.docx` will be used if a specific format file is missing.

# Placeholders
Use the following placeholders in your Word document:
- `{claimno}`
- `{tpaname}`
- `{insurancecompany}`
- `{claimtype}`
- `{patientname}`
- `{policyno}`
- `{allocationdate}`
- `{doa}`
- `{dod}`
- `{insuredname}`
- `{hospitalname}`
- `{hospitalhid}`
- `{hospitallocation}`
- `{hospitalstate}`
